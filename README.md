## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/brianwilsondev.git
```

2. Install dependencies:
```bash
cd brianwilsondev
npm install
```

3. Start the development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```
